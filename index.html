Learn CSS: Box Model

The Box Model is a CSS layout mechanism that the web browser uses to render content organized by box-shaped elements. Each element is made of four specific areas:

Element Definition CSS Property
Width and Height The width and height of the content area. width -or- height
Padding The amount of space between the content area and the border padding
Border The thickness and style of the border surrounding the content area and padding. border
Margin The amount of space bettwen the the border and outside edge of the element margin
